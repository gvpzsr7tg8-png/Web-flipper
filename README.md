# Web Flipper 🌐

A professional portfolio and services showcase website for web design and website flipping services.

## 🚀 Features

- **Responsive Design** - Mobile-friendly and works on all devices
- **Scroll Animations** - Smooth fade-in animations as you scroll
- **Pricing Packages** - Clear pricing tiers (Basic, Pro, Premium)
- **Social Proof** - 5-star reviews section
- **Portfolio** - Case studies showing before/after examples
- **Call-to-Action** - Direct payment links and contact information

## 📋 Sections

1. **Hero** - Eye-catching header
2. **About** - Personal bio and expertise
3. **Pricing** - Three service packages
4. **Reviews** - Customer testimonials
5. **Portfolio** - Example projects
6. **Contact** - Direct contact methods and payment link

## 🎨 Design

- Dark theme (#0b1220 background)
- Green accent buttons (#00d632 to #00ff88 gradient)
- Clean, modern typography
- Smooth animations on scroll

## 💻 How to Use

1. Open `index.html` in a web browser
2. Customize contact details as needed
3. Add your own reviews and portfolio examples
4. Update the Cash App link to your own
